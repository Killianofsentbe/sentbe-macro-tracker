# SentBe Macro Tracker — Static Mirror

거시 트래커(Global Market Intelligence) 시각화 페이지의 GitHub Pages 미러.
원본은 Google Apps Script 웹앱이며, 이 저장소는 구글 계정 제한 없이 열람 가능한 병행 채널입니다.

- `index.html` — 대시보드 (최신 브리핑·추이차트·히스토리·용어집)
- `detail/YYYY-MM-DD-am|pm.html` — 세션별 상세 페이지
- 갱신: Slack 신규 포스트 → GAS가 자동 커밋 (백필: build-static.js)
